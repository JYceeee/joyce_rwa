主要功能实现
  • MySQL数据集成：从数据库的product_details表获取所有产品数据
  • 多个卡片展示：为每个upcoming状态的项目创建独立卡片
  • 实时数据同步：使用数据库同步服务实现数据自动更新
  • 状态管理：加载状态、错误状态、空状态处理
  • 按钮位置：VIEW ALL PROJECTS按钮位于卡片下方
API接口调用

主要API调用：

GET /api/product_details

用于获取所有产品数据，HomeView会过滤出status为'upcoming'的项目

// 在HomeView.vue中的关键代码 async loadProducts() { const response = await productAPI.getAllProducts() if (response.status === 0) { this.products = response.data || [] } } // 过滤upcoming项目 upcomingProjects() { return this.products.filter(product => product.status === 'upcoming') }
数据库同步服务

实时数据更新：

  • 订阅产品列表更新事件
  • 检测新项目并显示通知
  • 自动刷新间隔30秒

用户界面状态

三种显示状态：

1. 加载状态：
<div v-if="loading" class="loading-state"> <div class="loading-spinner"></div> <p class="loading-text">正在加载项目数据...</p> </div>
2. 错误状态：
<div v-else-if="error" class="error-state"> <div class="error-icon">⚠️</div> <h4 class="error-title">加载失败</h4> <p class="error-description">{{ error }}</p> <button class="btn-primary" @click="loadProducts">重新加载</button> </div>
3. 项目卡片：
<article v-else-if="upcomingProjects.length > 0" v-for="project in upcomingProjects" class="listing-card"> <!-- 项目详情卡片 --> </article>

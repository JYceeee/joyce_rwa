1. 侧边栏更新

2. 路由配置

在 router/index.ts 中添加了FAQ路由：

import FAQView from '../views/info/FAQView.vue' { path: '/faq', name: 'faq', component: FAQView }

3. FAQ页面创建

创建了独立的FAQView.vue组件，包含：

4. 页面清理

从HomeView中移除了原有的FAQ section和相关CSS样式。

📊 Portfolio Analysis内联统计样式演示

🎯 样式优化说明

缩小字体并将标签和数值写在同一行，使统计卡片更加紧凑：

  • 内联布局: 标签和数值在同一行显示
  • 字体缩小: 标签字体从24px缩小到14px，数值字体从24px缩小到16px
  • 紧凑设计: 移除了图标，减少了垂直空间占用
  • 视觉层次: 通过颜色和字重区分标签和数值

📊 优化效果对比

❌ 原有样式问题:
  • 字体过大，占用过多空间
  • 图标占用额外空间
  • 垂直布局增加卡片高度
  • 卡片间距过大，浪费空间
  • 视觉层次不够清晰
✅ 新样式优势:
  • 字体适中，节省空间
  • 移除图标，更加简洁
  • 内联布局，减少高度
  • 缩小间距，更紧凑
  • 清晰的视觉层次

🔧 技术实现

HTML结构变化:


<div class="pf-analysis-stat-card">
  <div class="pf-stat-icon">📊</div>
  <div class="pf-stat-content">
    <div class="pf-stat-number">15</div>
    <div class="pf-stat-label">Total Transactions</div>
  </div>
</div>


<div class="pf-analysis-stat-card">
  <div class="pf-stat-content">
    <div class="pf-stat-label-inline">
      Total Transactions: <span class="pf-stat-number">15</span>
    </div>
  </div>
</div>

CSS样式变化:

 /* 一行四个统计卡片 */
 .pf-analysis-stats-four {
   display: grid;
   grid-template-columns: repeat(4, 1fr);  /* 一行四个 */
   gap: 8px;                               /* 缩小间距 */
   margin-bottom: 30px;
 }
 
 .pf-analysis-stat-card {
   padding: 16px;                          /* 缩小内边距 */
   gap: 8px;                               /* 缩小内部间距 */
 }
 
 .pf-stat-label-inline {
   font-size: 14px;           /* 标签字体 */
   color: var(--dark-muted);  /* 标签颜色 */
   line-height: 1.2;
 }
 
 .pf-stat-label-inline .pf-stat-number {
   font-size: 16px;           /* 数值字体(从24px缩小) */
   font-weight: 700;          /* 数值字重 */
   color: var(--dark-text);   /* 数值颜色 */
   margin-left: 4px;          /* 数值间距 */
 }
 
 /* 响应式设计 */
 @media (max-width: 1200px) {
   .pf-analysis-stats-four {
     grid-template-columns: repeat(2, 1fr);  /* 中等屏幕: 一行两个 */
   }
 }
 
 @media (max-width: 768px) {
   .pf-analysis-stats-four {
     grid-template-columns: 1fr;  /* 小屏幕: 一行一个 */
   }
 }

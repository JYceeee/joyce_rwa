🔧 技术实现

CSS修改:

.permission-badge {
  padding: 3px 8px;        /* 从 6px 12px 缩小 */
  border-radius: 12px;     /* 从 20px 缩小 */
  font-size: 11px;         /* 从 14px 缩小 */
  font-weight: 600;        /* 保持不变 */
  text-transform: uppercase; /* 保持不变 */
  letter-spacing: 0.3px;   /* 从 0.5px 缩小 */
}
